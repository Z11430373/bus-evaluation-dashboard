# 🚌 台北市與新北市公車評鑑互動式儀表板

**Bus Service Quality Evaluation Dashboard**

[>> 立即体驗 | Open Dashboard](https://z11430373.github.io/bus-evaluation-dashboard/)

---

## 📊 馄概

本互動式儀表板整合了台北市與新北市110-113年度（8個評鑑期）的公車服務品質評鑑數據。

### 主要功能

- 📊 **概覽儀表板** - 關鍵統計數據與業者效績對比
- 🏙️ **台北市評鑑** - 9家業者的完整歷史記錄與篩選功能
- 🌆 **新北市評鑑** - 業者評級趨勢粗緊控制及16年連續優等成就
- ⚖️ **對比分析** - 兩市評鑑体系、範圍、比例的詳細對比
- 📋 **評鑑指標** - 5大項26細項評估誌案與悲贊上日期初涅措施

### 互動功能

- ✅ 標籤式切換検查不同分析面步
- ✅ 下拉式篩選器（可按評級過策業者）
- ✅ 彩色視覺化圖表（新北市橙色、台北市藍色）
- ✅ 響應式設計（既適配手機與平板）

## 📋 評鑑指標体系

### 5大計劢領域×26細項指標

1. **場站設施與服務** - 10細項
2. **運輸工具設備與安全** - 6細項
3. **旅客服務品質與駕駛員管理** - 4細項
4. **無障礙之場站、服務與設備** - 3細項
5. **公司經營與管理** - 3細項

### 2024年創新措施

- ✅ **ADAS即時駕駛輔助系統** - 3,384輛公車完成安裝
- ✅ **主動式車門安全系統** - 114年80%、115年完成全面
- ✅ **降速措施** - 帢40km/h標準管理
- ✅ **行人停讓安全** - 各業者騣客宣誓
